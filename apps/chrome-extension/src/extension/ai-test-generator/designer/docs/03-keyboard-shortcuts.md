# 快捷键列表

使用快捷键可以大幅提高在可视化测试设计器中的工作效率。

## 文件操作

| 快捷键 | Mac | 功能 | 作用域 |
|--------|-----|------|--------|
| `Ctrl+S` | `Cmd+S` | 保存流程 | 全局 |
| `Ctrl+E` | `Cmd+E` | 导出 YAML | 全局 |
| `Ctrl+O` | `Cmd+O` | 打开项目 | 全局 |
| `Ctrl+N` | `Cmd+N` | 新建流程 | 全局 |

## 快捷键行为说明

### 作用域

- **全局**: 在设计器中随时可用
- **画布**: 仅在画布区域聚焦时可用

### 编辑元素例外

在以下元素中输入时，快捷键不会触发：

- 输入框（Input）
- 文本域（Textarea）
- 内容可编辑区域

### Modal 打开时

当 Modal 对话框打开时，大部分画布快捷键会被禁用，以避免干扰对话框操作。

## 自定义快捷键

快捷键可以通过修改配置文件自定义。具体方法请参考高级配置文档。

## 常用快捷键组合示例

### 快速创建流程

```
1. Ctrl+N          (新建流程)
2. 拖拽开始节点
3. 拖拽导航节点
4. Ctrl+C / Ctrl+V (复制粘贴节点)
5. 方向键调整位置
6. Ctrl+S          (保存)
```

### 快速编辑

```
1. Ctrl+A          (全选)
2. 方向键微调位置
3. Ctrl+Z          (撤销不满意的操作)
4. Ctrl+0          (适应视图查看整体)
5. Ctrl+E          (导出 YAML)
```
